## 데이터 수집
아래 경제지표들의 최신값을 신뢰할 수 있는 출처에서 직접 찾아오세요.
FRED, Bloomberg, Trading Economics, WSJ, Reuters 등 어디서든 가장 최신 데이터를 사용하세요.

수집할 지표:
- 미국 기준금리, 10년 실질금리, 10년 국채금리, M2 통화량, 연준 자산 규모
- 실질GDP 성장률, ISM 제조업 PMI, 비농업고용, 실업률, 주간 실업수당 청구
- Core PCE, CPI, 기대인플레이션(10Y), WTI 유가
- 장단기 금리차(10Y-2Y), 하이일드 스프레드, BBB 회사채 스프레드, 금융스트레스 지수
- VIX, 달러인덱스, S&P500, 나스닥, 원/달러 환율

## 분석 지시
- 각 지표의 현재 수준이 역사적·경제학적으로 어떤 의미인지 당신의 전문 지식으로 판단
- 정상/주의/위험 판정은 단순 수치 기준이 아니라 현재 경제 맥락을 함께 고려
- 여러 지표 간 상호작용과 복합 패턴을 파악
- 특이하거나 평소와 다른 변화에 주목

## 스타일 규칙

### 폰트
- 기본 폰트: Pretendard Variable (CDN 로드) → Inter → -apple-system → system-ui → sans-serif
- HTML 상단에 아래 CDN 링크를 삽입:
  ```
  <link href="https://cdn.jsdelivr.net/gh/orioncactus/pretendard@v1.3.9/dist/web/variable/pretendardvariable-dynamic-subset.min.css" rel="stylesheet">
  ```
- letter-spacing: -.01em 적용하여 깔끔한 가독성 확보

### 색상 규칙 (중요!)
변화의 방향(↑↓)이 아니라, 그 변화가 **경제·투자 관점에서 긍정/부정**인지에 따라 색상을 결정:
- **녹색(#16a34a)**: 경제·시장에 긍정적인 변화
- **빨간색(#dc2626)**: 경제·시장에 부정적인 변화
- **회색(#888)**: 중립 또는 변화 없음

예시:
| 지표 | 변화 | 판정 | 색상 |
|------|------|------|------|
| 10Y 국채금리 | ▼ 하락 | 금리 하락 = 주식에 긍정 | 녹색 |
| 실업률 | ▲ 상승 | 고용 악화 = 부정 | 빨간 |
| VIX | ▼ 하락 | 변동성 축소 = 긍정 | 녹색 |
| WTI 유가 | ▲ 상승 | 인플레 압력 = 부정 | 빨간 |
| CPI | ▼ 하락 | 인플레 완화 = 긍정 | 녹색 |
| 달러인덱스 | ▼ 하락 | 신흥국/원자재에 긍정 | 녹색 |
| M2 통화량 | ▲ 증가 | 유동성 확대 = 긍정 | 녹색 |
| 하이일드 스프레드 | ▲ 확대 | 신용위험 상승 = 부정 | 빨간 |
| 원/달러 | 원화 강세 | 외국인 유입 = 긍정 | 녹색 |
| Fear & Greed | 극단적 공포 | 심리 위축 = 부정 | 빨간 |

## 출력 구조 (HTML만, 앞뒤 설명 없이)

### 섹션 배치 순서 (중요!)
1. **헤더** — 경제 날씨 요약
2. **📌 오늘의 주목 신호** — 3가지 핵심 이벤트
3. **🎯 자산군별 투자 전략 매트릭스** — 4등급 분류 + 요약 바
4. **💡 오늘의 투자 시사점** — 종합 전략
5. **📋 핵심 지표 일람** — 상세 데이터 테이블 (맨 아래)

→ 요약·판단·전략이 상단, 상세 데이터가 하단에 위치

### 데이터 테이블 규칙
- 섹션 신호등 버튼 없음 (제거)
- 그룹명(유동성/성장/인플레이션/위험/시장심리): font-size .88rem, font-weight 700, border-bottom 2px solid [그룹색]
- 그룹 간 간격: margin-top 24px
- 4열 구조: 지표명(font-weight:600, 긍정/부정 색상) | 값(font-weight:700, 동일 색상) | 변화 설명(동일 색상) | 발표일 + 다음발표 + 예상값 (연한 회색 .62rem)
- 행 배경 없음. 텍스트 색상만으로 상태 표현
- 발표일 열 형식: `3/7 · 다음 4/4 · +80Ke` (직전발표일 · 다음발표일 · 예상값)
  - 매일 업데이트 지표는 `3/31 · 매일`로 표기

```html
<!-- 템플릿 구조 -->

  <!-- 1. 헤더 -->
  <div style="background:[색];border-left:4px solid [색];border-radius:8px;padding:16px 20px;margin-bottom:20px;">
    ...
  </div>

  <!-- 2. 주목 신호 3가지 -->
  <div style="margin-bottom:20px;">
    ...
  </div>

  <!-- 3. 자산군별 투자 전략 매트릭스 -->
  <!-- 4등급: 매우유리(#dcfce7) / 유리(#fefce8) / 중립(#f5f5f5) / 불리(#fef2f2) -->
  <div style="margin-bottom:20px;">
    <div style="font-size:.85rem;font-weight:700;margin-bottom:10px;">🎯 자산군별 투자 전략 매트릭스</div>
    <!-- 🟢 매우 유리 — 비중 확대 권장 -->
    <!-- 🟡 유리 — 비중 유지 -->
    <!-- ⚪ 중립 — 관망 또는 소량 분할매수 -->
    <!-- 🔴 불리 — 비중 축소 또는 회피 -->
    <!-- 하단 요약 바: 등급별 자산군을 한눈에 보여주는 색상 바 -->
  </div>

  <!-- 4. 투자 시사점 -->
  <div style="background:#f8f8f8;border-radius:8px;padding:14px 18px;margin-bottom:24px;">
    ...
  </div>

  <!-- 5. 핵심 지표 일람 (맨 아래) -->
  <!-- 그룹명: .88rem 굵게 + border-bottom 2px + 그룹 간 margin-top:24px -->
  <div style="margin-bottom:16px;">
    <div style="font-size:.85rem;font-weight:700;">📋 핵심 지표 일람</div>
    <!-- 그룹별 테이블 반복 -->
    <div style="font-size:.88rem;font-weight:700;color:[그룹색];margin:24px 0 8px;padding-bottom:4px;border-bottom:2px solid [그룹색];">[이모지] [그룹명]</div>
    <table>
      <tr style="border-bottom:1px solid #eee;">
        <td style="font-weight:600;color:[긍정/부정/중립색];">[지표명]</td>
        <td style="font-weight:700;color:[동일색];">[값]</td>
        <td style="color:[동일색];font-size:.7rem;">[변화]</td>
        <td style="color:#bbb;font-size:.62rem;">[발표일 · 다음 M/DD · 예상값e]</td>
      </tr>
    </table>
  </div>

</div>
```
