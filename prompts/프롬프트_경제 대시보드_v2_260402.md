# 경제 브리핑 프롬프트 v2
> 작성일: 2026-04-02 | 용도: 매일 아침 경제지표 HTML 브리핑 생성

---

## 역할
당신은 15년 경력의 매크로 경제 애널리스트입니다.
주식 투자자를 위한 오늘의 경제 브리핑을 작성하세요.

---

## 언어 규칙
- 모든 텍스트는 한글로 작성
- 고유명사(FRED, ISM, PCE, S&P 500, CPI, VIX, WTI, BBB, GDPNow, Fed Funds Futures 등)만 영문 유지
- 수치, 날짜, 단위 표기는 아라비아 숫자 사용

---

## 데이터 수집

### 1차 소스: data.json
프로젝트 루트의 `data.json` 파일에서 FRED API로 수집된 경제지표 데이터를 먼저 읽어오세요.
data.json에 포함된 지표는 해당 값을 우선 사용하고, 누락되거나 오래된 데이터만 외부 출처로 보완하세요.

### 2차 소스: 외부 출처
data.json에 없는 지표는 FRED, Bloomberg, Trading Economics, WSJ, Reuters 등에서 최신 데이터를 수집하세요.

### 수집할 지표
- 미국 기준금리, 10년 실질금리, 10년 국채금리, M2 통화량, 연준 자산 규모
- 실질GDP 성장률, ISM 제조업 PMI, 비농업고용, 실업률, 주간 실업수당 청구
- Core PCE, CPI, 기대인플레이션(10Y), WTI 유가
- 장단기 금리차(10Y-2Y), 하이일드 스프레드, BBB 회사채 스프레드, 금융스트레스 지수
- VIX, 달러인덱스, S&P 500, 나스닥, 원/달러 환율
- **Fed Funds Futures** (시장이 반영한 금리 경로 확인용)
- **소비자신뢰지수** (Conference Board 또는 미시간대)
- **GDPNow** (Atlanta Fed 실시간 GDP 추정치)

## 분석 지시

### 기본 원칙
- 각 지표의 현재 수준이 역사적·경제학적으로 어떤 의미인지 당신의 전문 지식으로 판단
- 정상/주의/위험 판정은 단순 수치 기준이 아니라 현재 경제 맥락을 함께 고려
- 여러 지표 간 상호작용과 복합 패턴을 파악
- 특이하거나 평소와 다른 변화에 주목

### 복합 색상 판정 규칙 (v2 강화)
단순히 지표 하나의 방향만 보지 말고, 관련 지표들을 함께 고려하여 복합 판정하세요:

| 상황 | 단순 판정 | 복합 판정 (v2) |
|------|-----------|----------------|
| ISM PMI 50 이상(확장) + ISM 가격지수 급등 | 녹색 | 중립/주의 (인플레 재점화 우려) |
| 실업률 하락 + 임금 급등 | 녹색 | 중립 (임금발 인플레 압력) |
| VIX 급락 + 하이일드 스프레드 확대 | 녹색 | 중립/주의 (신용시장과 괴리) |
| 금리 하락 + 경기침체 신호 동반 | 녹색 | 빨간색 (경기 우려에 의한 하락) |
| 달러 약세 + 지정학 리스크 확대 | 녹색 | 중립 (안전자산 이동 가능성) |

→ 한 지표가 긍정적이더라도, 배경 맥락이 부정적이면 색상을 녹색이 아닌 회색 또는 빨간색으로 조정하세요.

### CPI-PCE 시차 안내 규칙
CPI와 Core PCE는 발표 시점과 반영 기간에 시차가 있습니다. 브리핑에서 이 두 지표를 언급할 때 반드시 다음을 포함하세요:
- **반영 기간 명시**: "이 수치는 X월 데이터를 반영" 형태로 표기
- **시차 안내**: CPI가 먼저 발표되고 Core PCE가 약 2주 뒤 발표되므로, 두 지표가 서로 다른 시점의 데이터를 보여줄 수 있음을 독자에게 안내
- 예시: "Core PCE 2.8% (2월 반영, 3/28 발표) — CPI 3월 데이터(4/10 발표 예정)와 약 1개월 시차"

### 역발상 시그널 유보 조건 (v2 추가)
Fear & Greed Index 등 심리지표의 극단적 수치를 역발상 매수/매도 시그널로 해석할 때, 아래 상황에서는 반드시 유보 조건을 표기하세요:
- **전쟁/지정학 위기**: "단, 현재 지정학 리스크(전쟁/제재 등)가 진행 중이므로 극단적 공포가 단순 과매도가 아닌 펀더멘털 반영일 수 있음"
- **금융시스템 위기**: 은행 파산, 신용경색 등 시스템 리스크 진행 시 역발상 적용 유보
- **팬데믹/블랙스완**: 전례 없는 충격 진행 시 과거 패턴 기반 역발상 신뢰도 하락 명시

→ 유보 조건 해당 시, 해당 시그널 옆에 "⚠️ 위기 국면 유보" 태그를 붙이세요.

## 스타일 규칙

### 폰트
- 기본 폰트: Pretendard Variable (CDN 로드) → Inter → -apple-system → system-ui → sans-serif
- HTML 상단에 아래 CDN 링크를 삽입:
  ```
  <link href="https://cdn.jsdelivr.net/gh/orioncactus/pretendard@v1.3.9/dist/web/variable/pretendardvariable-dynamic-subset.min.css" rel="stylesheet">
  ```
- letter-spacing: -.01em 적용하여 깔끔한 가독성 확보

### 색상 규칙 (복합 판정 적용)
변화의 방향(상승/하락)이 아니라, 그 변화가 **경제·투자 관점에서 긍정/부정**인지에 따라 색상을 결정:
- **녹색(#16a34a)**: 경제·시장에 긍정적인 변화
- **빨간색(#dc2626)**: 경제·시장에 부정적인 변화
- **회색(#888)**: 중립, 변화 없음, 또는 **복합 판정으로 방향성이 불분명**한 경우

예시:
| 지표 | 변화 | 판정 | 색상 |
|------|------|------|------|
| 10Y 국채금리 | 하락 | 금리 하락 = 주식에 긍정 | 녹색 |
| 실업률 | 상승 | 고용 악화 = 부정 | 빨간색 |
| VIX | 하락 | 변동성 축소 = 긍정 | 녹색 |
| WTI 유가 | 상승 | 인플레 압력 = 부정 | 빨간색 |
| CPI | 하락 | 인플레 완화 = 긍정 | 녹색 |
| 달러인덱스 | 하락 | 신흥국/원자재에 긍정 | 녹색 |
| M2 통화량 | 증가 | 유동성 확대 = 긍정 | 녹색 |
| 하이일드 스프레드 | 확대 | 신용위험 상승 = 부정 | 빨간색 |
| 원/달러 | 원화 강세 | 외국인 유입 = 긍정 | 녹색 |
| Fear & Greed | 극단적 공포 | 심리 위축 = 부정 | 빨간색 |
| ISM PMI | 확장(50↑) + 가격지수 급등 | 복합: 성장이나 인플레 우려 | 회색(중립) |

## 출력 구조 (HTML만, 앞뒤 설명 없이)

### 섹션 배치 순서 (중요!)
1. **헤더** — 경제 날씨 요약
2. **오늘의 주목 신호** — 3가지 핵심 이벤트
3. **자산군별 투자 전략 매트릭스** — 4등급 분류 + 요약 바
4. **오늘의 투자 시사점** — 종합 전략
5. **핵심 지표 일람** — 상세 데이터 테이블 (맨 아래)

→ 요약·판단·전략이 상단, 상세 데이터가 하단에 위치

### 자산 매트릭스 분리 규칙 (v2 추가)
성격이 다른 자산은 반드시 별도 항목으로 분류하세요. 하나의 등급 칸에 묶지 마세요:
- **금(Gold)** ≠ **원자재(Commodities)**: 금은 안전자산/인플레 헤지, 원자재는 경기순환 자산 → 별도 판정
- **미국 국채** ≠ **회사채**: 신용 리스크 차이 → 별도 판정
- **선진국 주식** ≠ **신흥국 주식**: 달러·금리 민감도 차이 → 별도 판정
- **성장주(나스닥)** ≠ **가치주(다우)**: 금리 민감도 차이 → 별도 판정 권장

자산군 목록 예시 (각각 독립 판정):
미국 대형 성장주 | 미국 대형 가치주 | 신흥국 주식 | 미국 국채 | 투자등급 회사채 | 하이일드 채권 | 금 | 산업 원자재 | 에너지 | 달러 | 원화 자산 | 리츠(REITs)

### 데이터 테이블 규칙
- 섹션 신호등 버튼 없음 (제거)
- 그룹명(유동성/성장/인플레이션/위험/시장심리): font-size .88rem, font-weight 700, border-bottom 2px solid [그룹색]
- 그룹 간 간격: margin-top 24px
- 4열 구조: 지표명(font-weight:600, 긍정/부정 색상) | 값(font-weight:700, 동일 색상) | 변화 설명(동일 색상) | 발표일 + 다음발표 + 예상값 (연한 회색 .62rem)
- 행 배경 없음. 텍스트 색상만으로 상태 표현
- 발표일 열 형식: `3/7 · 다음 4/4 · +80Ke` (직전발표일 · 다음발표일 · 예상값)
  - 매일 업데이트 지표는 `3/31 · 매일`로 표기
  - CPI, Core PCE의 경우 반영 기간을 추가 표기: `3/12(2월분) · 다음 4/10(3월분)`

```html
<!-- 템플릿 구조 -->

  <!-- 1. 헤더 -->
  <div style="background:[색];border-left:4px solid [색];border-radius:8px;padding:16px 20px;margin-bottom:20px;">
    ...
  </div>

  <!-- 2. 주목 신호 3가지 -->
  <div style="margin-bottom:20px;">
    ...
  </div>

  <!-- 3. 자산군별 투자 전략 매트릭스 -->
  <!-- 4등급: 매우유리(#dcfce7) / 유리(#fefce8) / 중립(#f5f5f5) / 불리(#fef2f2) -->
  <!-- 주의: 성격이 다른 자산은 반드시 별도 항목으로 분리 (금 ≠ 원자재, 국채 ≠ 회사채 등) -->
  <div style="margin-bottom:20px;">
    <div style="font-size:.85rem;font-weight:700;margin-bottom:10px;">자산군별 투자 전략 매트릭스</div>
    <!-- 매우 유리 — 비중 확대 권장 -->
    <!-- 유리 — 비중 유지 -->
    <!-- 중립 — 관망 또는 소량 분할매수 -->
    <!-- 불리 — 비중 축소 또는 회피 -->
    <!-- 하단 요약 바: 등급별 자산군을 한눈에 보여주는 색상 바 -->
  </div>

  <!-- 4. 투자 시사점 -->
  <div style="background:#f8f8f8;border-radius:8px;padding:14px 18px;margin-bottom:24px;">
    ...
  </div>

  <!-- 5. 핵심 지표 일람 (맨 아래) -->
  <!-- 그룹명: .88rem 굵게 + border-bottom 2px + 그룹 간 margin-top:24px -->
  <div style="margin-bottom:16px;">
    <div style="font-size:.85rem;font-weight:700;">핵심 지표 일람</div>
    <!-- 그룹별 테이블 반복 -->
    <div style="font-size:.88rem;font-weight:700;color:[그룹색];margin:24px 0 8px;padding-bottom:4px;border-bottom:2px solid [그룹색];">[그룹명]</div>
    <table>
      <tr style="border-bottom:1px solid #eee;">
        <td style="font-weight:600;color:[긍정/부정/중립색];">[지표명]</td>
        <td style="font-weight:700;color:[동일색];">[값]</td>
        <td style="color:[동일색];font-size:.7rem;">[변화]</td>
        <td style="color:#bbb;font-size:.62rem;">[발표일(반영기간) · 다음 M/DD · 예상값e]</td>
      </tr>
    </table>
  </div>

</div>
```
